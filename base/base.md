Um sistema operacional de demandas para design, que resolve 4 dores principais:
você esquece demandas

retrabalha ideias
👉 Esse sistema elimina tudo isso com automação + visão clara + execução simples

🧩 ESTRUTURA GERAL (como tudo se conecta)
O sistema tem 4 partes que funcionam juntas:
Entrada rápida (captura)
Base central (planilha inteligente)
Execução visual (kanban)
Análise (dashboard)
Tudo sincronizado. Nada duplicado. Nada manual além do essencial.

⚙️ FLUXO REAL DE USO (seu dia a dia)
Você aperta um atalho e cria uma demanda em segundos
O sistema organiza automaticamente (tipo, título, prioridade)
A demanda cai no Inbox
Você arrasta para “Hoje” quando for executar
Finaliza e marca como concluído
Isso alimenta histórico e gráficos automaticamente
👉 Sem planilha manual, sem controle paralelo

🧱 1. CAPTURA RÁPIDA (entrada mínima)
🎯 Objetivo
Registrar sem pensar
Como funciona
Um campo simples com dois inputs:
Cliente
Demanda
👉 Só isso é obrigatório

💡 Experiência ideal
Você abre com atalho (tipo Ctrl + N)
Digita em 3 segundos
Aperta Enter
Pronto

🤖 O que o sistema faz automaticamente
Assim que você salva, ele:
cria um título curto (ex: “AM - Promo Whey”)
identifica o tipo de demanda (Feed, Story, Vídeo…)
define uma prioridade
sugere um prazo
👉 você não precisa pensar nisso na entrada

📊 2. BASE CENTRAL (planilha inteligente)
🎯 Objetivo
Ser o cérebro do sistema

Como ela é organizada
IDENTIFICAÇÃO
Status (Inbox, Hoje, Depois, Concluído, Travado)
Cliente
Tipo
Título curto

CONTEÚDO
Demanda original (o que você escreveu)
Observações
Referências (opcional)

EXECUÇÃO
Prioridade
Prazo
Responsável (se tiver equipe)

CONTROLE
Data de criação
Data de conclusão

🔁 Regra fundamental
👉 Tudo nasce aqui
 👉 Tudo atualiza aqui
 👉 Nada existe fora daqui

🧩 3. KANBAN (execução real)
🎯 Objetivo
Te mostrar exatamente o que fazer

Estrutura simples (sem complicar)
Inbox → tudo que entrou
Hoje → o que você vai fazer
Depois → backlog leve
👉 Só isso. Mais colunas só atrapalham.

Como você usa
Criou → vai pro Inbox
Vai trabalhar → arrasta pra Hoje
Terminou → marca como concluído

💡 Importante
Marcar no kanban atualiza a planilha
Alterar na planilha move no kanban
👉 tudo sincronizado automaticamente

🔍 4. BUSCA RÁPIDA
🎯 Objetivo
Encontrar qualquer job em segundos

Como funciona

Você digita qualquer palavra
Ele busca em:
cliente
título
descrição

Experiência ideal
Busca instantânea (sem botão)
Atalho tipo Ctrl + K
Resultado em tempo real
